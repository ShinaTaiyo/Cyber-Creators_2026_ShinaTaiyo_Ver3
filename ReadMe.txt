◆このゲームについて
・・・
TPSアクションゲームです。ショットとダイブという二つの技を駆使して縦横無尽に駆け回り、敵を効率的に駆逐しよう！

◆起動方法
・・・
exeファイルをダブルクリック


◆基本操作
・・・
・チュートリアルがあります。そこで覚えてください。

コントローラー
A：ジャンプ
X：モード切替（射撃モードとダイブモードを切り替えます）
Y：後ろを向く
右スティック：移動
左スティック：視点移動
Rトリガー：（射撃モードなら→射撃、ダイブモードなら→ダイブ）
Lトリガー：ダッシュ
STARTボタン：画面遷移

◆実装内容
・・・
・通常敵を２体実装（射撃に弱い敵、ダイブに弱い敵）。
・一度置いたオブジェクトを再度自由に値を変えることができるエディタを実装。
・ゲームの大まかな流れを実装。
・フェーズマネージャー実装
・モーションクラス実装
・ダイブやカメラの角度調整を完了済み。

◆これから取り組む内容
・ステージギミックを実装。ダッシュやダイブ、射撃などのアクションを楽しく使えるステージを作る。
・ボスを実装。
